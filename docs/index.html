REST API for Real Estate rental data, a spatial Bayesian modeling approach with INLA.

REST API for Real Estate rental data, a spatial Bayesian modeling approach with INLA.

Niccolò Salvini

Last compiled on 12 novembre, 2020

Preliminary Content

Abstract

This thesis has the aim to estimate and predict monthly price for rents in Milan Real Estate exploiting geostatistical data component. Data originates from immobiliare.it database and it is extracted through customized scraping techniques in parallel computing. Scraping functions are wrapped into a API. The starting building framework is R Plumber. Then the API infrastructure passes from Containerizing through docker to being served in a AWS EC2 instance secured and load balanced with NGINX. Furthermore a scheduler programs dailuy extraction and subsequently stores data in a MongoDB ATLAS database. The integration of these technologies makes the API infrastructure REST. The spatial component is treated as a GP and falls under the umbrella of point referenced data modeling, so dedicated techniques are required. A hierarchical Bayesian setting on a Hedonic price model is chosen since it can easily handle different parameter and hyper parameter levels and enables interpretability. Integrated Nested Laplace Approximation methodology reduces the GP spatial process to a GMRF representation of the surface through Stochastic Partial Differential Equation SPDE. The benefit is a sparse matrix representation of the process for which can be used numerical methods. Price are then kriged on a surface and are evaluated into a shiny app back end. Visualization are made with a combination of ggplot2 and leaflet following a Tidy approach.
